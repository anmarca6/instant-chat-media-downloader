Instant Chat Media Downloader

v1.9.3

How to Use

  1. Open a conversation in WhatsApp Web
  2. Click the floating download button
  3. Select a time range (1h, 3h, or 24h)
  4. Click "Magic Scan" to detect media
  5. Click "Download All" to save the files

Features

  • Magic Scan - One-click scanning of visible messages
  • Time-based filtering (1h, 3h, 24h)
  • Bulk download images and documents
  • Auto-organized folders by date/time
  • Files organized by conversation
  • Completely local and secure

Important

This extension only works on web.whatsapp.com. Make sure you have a conversation open before using the extension.

All files are downloaded directly to your browser's configured downloads folder.
